2nd International
Semantic Web Conference
(ISWC2003)

20-23 October 2003

Sundial Resort, Sanibel Island, Florida, USA
supported by the Semantic Web Science Association in cooperation with the OntoWeb network, the
DARPA program,
and our key corporate sponsor

ISWC is a major international forum at which research on all aspects of the Semantic Web is presented. ISWC2003 follows the 1st International Semantic Web Conference (ISWC2002) which was held in Sardinia, Italy, 9-12 June 2002.

ISWC2003 will be held in Florida (USA) from 20-23 October 2003, in conjunction with K-CAP 2003: The Second International Conference on Knowledge Capture.

 
Tutorials

A day of Semantic Web tutorials is planned for Monday, 20 October 2003.

The following tutorials have been accepted following the Call for Tutorials:

  • Agent-Mediated Semantic Web/Grid Services (full day)
    Katia Sycara and Terry Payne
  • Tutorial on OWL (morning)
    Peter F. Patel-Schneider, Ian Horrocks, and Sean Bechhofer
  • Creating Ontologies and Semantic Web Applications with Protégé (afternoon - may be taken alone or as a logical continuation of the Tutorial on OWL)
    Holger Knublauch and Natasha F. Noy
  • Information Integration on the World Wide Web (4 hours)
    Heiner Stuckenschmidt, Ubbo Visser, Holger Wache

Conference

ISWC2003 will include three days of technical sessions related to all aspects of the Semantic Web. The program includes both technical and survey/overview papers, as well as descriptions of working Semantic Web systems, position statements, and reports on work in progress.

The Program lists the papers accepted following the Call for Papers.


Workshops

A day of Semantic Web workshops is planned for Monday, 20 October 2003.

The following workshop proposals have been accepted following the Call for Workshop Proposals:

  • Practical and Scalable Semantic Systems
  • Semantic Integration
  • Semantic Web Technologies for Searching and Retrieving Scientific Data
  • Human Language Technology for the Semantic Web and Web Services
  • Rules and Rule Markup Languages for the Semantic Web
  • Evaluation of Ontology-Based Tools

Venue and Transportation

ISWC2003 will be held at the Sundial Resort on beautiful Sanibel Island.

Hotel reservations can be made online. Enter the promotion code ISWC to get the conference rate of $125/night (1-bedroom) or $230/night (2-bedroom). These rates are available for reservations made by 4 September 2003.

The Sundial Resort is full as of September 21. The Sanibel Inn is within walking distance. The Holiday Inn is also nearby.

Major airports serving Sanibel include Ft. Myers (RSW - about 15 miles away), Tampa (TPA - about 150 miles), and Miami (MIA - about 170 miles). From Ft. Myers, the hotel recommends Sanibel Taxi (phone: +1 239-472-4160). Costs are approximately $40 + $15/person over 2 people.

The hotel check-out time is 11am day of departure.
